## Notes de conception

- **Palette** : Fond sombre (#1a1a2e, #16213e) pour le dashboard, fond clair (#f8f9fa) pour GestionDeTâches
- **Icônes** : Emojis unicode pour les actions et statuts (🔵🟡🟢), compatibles navigateur
- **Navigation** : Barre supérieure avec nom d'utilisateur et déconnexion
- **Responsive** : Tableaux adaptables, colonnes prioritaires masquables sur mobile
- **Accessibilité** : Contraste élevé, libellés explicites
